Quick Start
-----------

.. code-block:: bash

   # Install GRA Core Platform v5.5 LTS
   pip install gra-core==5.5.0
   
   # Initialize LTS project
   gra-core init my-project --lts
   
   # Start with LTS configuration
   gra-core serve --config=lts
