Development Setup
-----------------

.. code-block:: bash

   # Clone development repository
   git clone https://github.com/bankofamerica/gra-core.git
   cd gra-core
   
   # Switch to development branch
   git checkout develop
   
   # Install development dependencies
   pip install -e .[dev]
   
   # Run in development mode
   gra-core serve --dev
