Quick Start
-----------

Get started with GRA Core Platform v5.6:

.. code-block:: bash

   # Install GRA Core Platform v5.6
   pip install gra-core==5.6.0
   
   # Initialize your project
   gra-core init my-project --version=5.6
   
   # Start development server
   gra-core serve
