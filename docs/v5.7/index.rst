Quick Start
-----------

Get started with GRA Core Platform v5.7:

.. code-block:: bash

   # Install GRA Core Platform v5.7
   pip install gra-core==5.7.0
   
   # Initialize your project
   gra-core init my-project
   
   # Start development server
   gra-core serve

Key Components
--------------

* **GCP Integration**: Seamless integration with Google Cloud Platform services
* **API Gateway**: Centralized API management and routing
* **Authentication**: OAuth 2.0 and JWT-based authentication

* **Monitoring**: Built-in monitoring and alerting system
