Quick Start
-----------

Get started with GRA Core Platform in just a few minutes:

1. **Install the Platform**

   .. code-block:: bash

      pip install gra-core-platform==5.6.0

2. **Initialize Your Project**

   .. code-block:: bash

      gra-core init my-project
      cd my-project

3. **Start Development Server**

   .. code-block:: bash

      gra-core serve

4. **Open in Browser**

   Navigate to http://localhost:8000

Prerequisites
-------------

* Python 3.8 or higher
* Node.js 16 or higher
* Docker (optional)
